# Geometryx Posters

Static hosting for [Geometryx](https://geometryx.io) metro Instagram posters.

These JPEGs are fetched by [Buffer](https://buffer.com) when scheduled Instagram posts publish to [@geometryx.us](https://instagram.com/geometryx.us).

**Format:** 1086×1448 JPEG (3:4 aspect, Instagram feed ratio), ~550 KB each.

## URLs

```
https://raw.githubusercontent.com/Carson2113/geometryx-posters/main/<slug>.jpg
```
